# Applied Machine Learning Portal

A comprehensive web-based Applied Machine Learning Portal developed using **Python, Flask, Scikit-Learn, TensorFlow, and Hugging Face Transformers**. The project integrates Machine Learning, Deep Learning, Generative AI, and Association Rule Mining algorithms into a single interactive dashboard.

---

## Features

### Machine Learning

* Naive Bayes (Spam Detection)
* K-Nearest Neighbors (Flower Classification)
* Linear Regression
* K-Means Clustering
* DBSCAN Clustering
* One-R Rule-Based Classification

### Deep Learning

* Artificial Neural Network (ANN)
* Convolutional Neural Network (CNN)
* LSTM Sentiment Analysis

### Generative AI

* Sentiment Analysis
* English to Urdu Translation
* Speech-to-Text
* Voice Question Answering
* Named Entity Recognition (NER)
* Text Generation
* Zero-Shot Classification

### Association Rule Mining

* Apriori Algorithm
* Frequent Itemsets and Rule Generation

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Applied-Machine-Learning-Project.git
```

Move to project directory:

```bash
cd Applied-Machine-Learning-Project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
